## Syncra
AI-powered event planning for student clubs and volunteer-led communities. Organizers import an event brief and get a structured, editable plan in under 10 minutes. Volunteers get a clean mobile view of exactly what they need to do.

# What it does

Import an event brief from a PDF, Notion page, or manual text input
AI extracts tasks, contacts, timing, and locations using Gemini 2.5 Flash
Organizers review and edit the generated plan before publishing
Volunteers see a personalized, mobile-first view of their assigned tasks and a "what changed" feed
